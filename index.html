WHAT IS BUSY WORK MAN?

BUSY WORK MAN is a collectible pixel art collection of 3000 randomly generated characters. Each character is created through algorithm with 7 randomly changing properties: skin, hair, eyes, face, shirt, pants, shoes. Among those 3000 characters, there are 30 with unique background design which is not programmed.

HOW IS BUSY WORK MAN GENERATED?

Random

  • style
  • color
  • background
  • TRAIT??

Each BUSY WORK MAN is generated through a program with unique design. The program first creates the pixel art, then it compares the new art piece with existed BWMs through an image similarity algorithm. After the program validates there is no duplicated BWM in the system, it assigns a random number between 1 ~ 2970 (2971 ~ 3000: Special BWM) to the art piece.

TRAITS LIMIT

There are some rare traits that have a maximum limit. There will be no more BWM generated with that trait if the maximum is reached. These traits will be revealed as we progress.

CONSTANT TRAIT: Each of the background color has a maximum limit, once the limit is reached, it will be removed from selection pool.
